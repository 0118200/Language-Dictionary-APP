# Language-Dictionary-ANDROID-

This apk is dictionary Indonesian (input) to give an English (output), this apk is for study Indonesian people to learn English language. With in this English Language we can also understand definitions, pronounce, and anything else. We used 2 API for communication between translate and dictionary, at the first time we built but those input must be an english, but we tought user is don't know english language, and then we development this APK to had a 2 API, 1 for english language dictionary, 1 more for translate. And then we had a input from Indonesian language to get an English language dictionary for output

## Specification

This program is used java to built an APK, and we used Android Studio to development English Language Dictionary
Program Language : java
API : English Language Dictionary API, Translate API
